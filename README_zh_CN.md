# Riff

[English](README.md)

## 💡 简介

[Riff](https://github.com/siyuan-note/riff) 是思源笔记的间隔重复组件。

状态：**开发中**

## ✨ 特性

* 支持扩展算法
  * [FSRS](https://github.com/open-spaced-repetition/free-spaced-repetition-scheduler)
  * SM2
* 支持数据持久化
